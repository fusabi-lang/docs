# Welcome to Fusabi Lang Documentation

This is the central documentation hub for all projects in the fusabi-lang organization.

## Overview

This documentation site automatically aggregates and publishes documentation from all public repositories in the fusabi-lang GitHub organization. Each project maintains its own documentation in its respective repository, and changes are synchronized daily.

## Features

- **Automatic Synchronization**: Daily updates from all tracked repositories
- **Full-Text Search**: Quickly find what you're looking for
- **Graph View**: Visualize relationships between documentation pages
- **Backlinks**: See which pages reference the current page
- **Dark Mode**: Toggle between light and dark themes

## Getting Started

Browse the documentation by:
- Using the search bar above
- Exploring the sidebar navigation
- Viewing the [[content/projects/index|Projects]] page for a complete list
- Checking out the graph view to see how documentation is connected

## About Fusabi Lang

Fusabi Lang is an organization focused on building programming language tools and related projects. Visit our [GitHub organization](https://github.com/fusabi-lang) to learn more.

## Contributing

Each project maintains its own documentation in its `/docs` directory. To contribute:
1. Find the project repository you want to document
2. Add or edit markdown files in the `/docs` directory
3. Submit a pull request
4. Your changes will be automatically synchronized to this site

## Technical Details

This site is built with:
- **Quartz v4**: Static site generator for Obsidian vaults
- **Nushell**: Automation scripts for repository discovery and synchronization
- **GitHub Actions**: Continuous integration and deployment
- **Git Submodules**: Links to documentation from each repository

For more information about the technical architecture, see [[CLAUDE]].
